
## 快速上手

```bash
# clone the project
git clone https://blog.csdn.net/qq_40739917/vuepress-theme-vdoing.git

# enter the project directory
cd vuepress-theme-vdoing

# install dependency
npm install # or yarn install

# develop
npm run dev # or yarn dev
```
