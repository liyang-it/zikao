## 1.(单选题)下列选项中，属于公司治理结构的有（ ）。
* A 欧美模式
* B 日美欧洲大陆模式
* C “共同治理”模式
* D 以上三个都是

正确答案：**`C`**

>解析
>
>公司治理结构通常有英美模式、日本欧洲大陆模式等。股东治理”模式又称为英美模式。股东治理”模式是单层制模式，也称为一元制模式。共同治理”模式又称为日本欧洲大陆模式。共同治理”模式是双层制模式，也称二元制模式。

## 2.(单选题)关于利益相关者，下列说法错误的是（ ）。
* A 所有利益相关者都与公司存在交易关系，在交易中，每一个群体都希望自己的利益得到满足
* B 公司在制定企业战略时，必须要考虑到利益相关者的权益主张，否则，利益相关者会撤销他们对公司的支持
* C 利益相关者能够影响企业的战略决策，他们的意见一定要作为企业战略决策时需要考虑的因素
* D 不同的利益相关者对企业战略决策的影响程度是不同的，根据影响程度的不同，我们将其分为直接利益相关者和间接利益相关者

正确答案：**`D`**

>解析
>
>不同的利益相关者对企业战略决策的影响程度是不同的，根据影响程度的不同，我们将其分为主要利益相关者和次要利益相关者。因为与利益的密切关系不同，我们又将其分为直接利益相关者和间接利益相关者，故D选项错误。

## 3.(单选题)下列关于“股东治理”模式说法正确的是（ ）。
* A 此模式又称为日本欧洲大陆模式
* B 在企业中的组织是以平等的个人契约为基础
* C 倡集体主义，注重劳资的协调
* D 其目的是旨在发挥利益相关者的作用

正确答案：**`B`**

>解析
>
>“股东治理”模式又称为英美模式。英美等国重视个人主义的不同思想，在企业中的组织只是以平等的个人契约为基础。共同治理”模式又称为日本欧洲大陆模式。日本和欧洲大陆尊重人权，在企业的经营中，提倡集体主义，注重劳资的协调。其目的旨在发挥利益相关者的作用。

## 7.(单选题)在SWOT模型中，具有一定的外部机会，但企业内部却出现资源危机，则该企业应采取（ ）。
* A 增长型战略
* B 多种经营战略
* C 扭转型战略
* D 防御型战略

正确答案：**`C`**

>解析
>
>SWOT分析中第三步是战略匹配和选择，根据标准的SWOT分析得出在OS、OW、TS和TW四种条件下的战略选择。首先要理解四种战略的含义。针对OS匹配适宜选择的战略：在拥有外部机会和内部优势的条件下，常采用的战略选择是增长战略。具体表现为：增加资源投入，巩固竞争优势，扩大对机会的利用，必要时需要企业在一定程度上集中所能支配的资源等。针对OW匹配适宜选择的战略：在拥有外部机会和内部劣势的条件下，企业应采取扭转型战略。针对TS匹配适宜选择的战略：在拥有外部威胁和内部优势的条件下，企业应采取多种经营战略。针对TW匹配适宜选择的战略：在处于外部威胁和内部劣势的条件下，企业应采取防御型战略。

## 8.(单选题)在SWOT分析中，代表企业具有外部市场机会的是（ ）。
* A S
* B W
* C O
* D T

正确答案：**C**

>解析
>
>SWOT分析法通过对外部环境条件和内部资源能力的分析，进行系统评价，来指导企业进行战略选择和制定。其中，S：Strengths，是指企业所擅长的方面，或所特有的能够提高竞争力的能力。W：Weaknesses，是指企业缺少或做得不好的方面，或某种会使企业处于劣势的条件。O：OPPortunities，是企业具有的外部市场机会。T：Threats，是企业面临的外部威胁。

## 9.(单选题)下列选项中，属于SWOT分析中企业外部威胁的是（ ）。
* A 市场进入壁垒降低
* B 关键领域里竞争能力正在丧失或很弱
* C 客户群的扩大趋势或产品细分市场
* D 市场增长率下降

正确答案：**`D`**

>解析
>
>企业面临的外部威胁的主要表现形式有：
>（1）强大的新竞争对手很可能会进入市场；
>（2）替代品抢占公司销售额；
>（3）市场增长率下降；
>（4）外汇汇率和外国政府贸易政策的不利变动；
>（5）有关部门所采取的管理措施会使公司付出很大的代价；
>（6）很容易受到经济萧条和业务周期的冲击等；
>（7）客户或供应商的谈判能力会提高；
>（8）购买者需求和品位朝偏离行业的方向变动；
>（9）不利的人口特征的变动；
>（10）容易受到行业驱动因素的冲击。

## 11.(单选题)下列选项中，属于战略管理分析方法中投资组合分析法的是（ ）。
* A 波士顿矩阵分析法
* B SWOT分析法
* C 波特五力模型
* D 宏观环境分析法

正确答案：**`A`**

>解析
>
>投资组合分析法是多元化企业制定公司战略时最受欢迎的工具之一，最常用的两种组合分析方法是波士顿矩阵分析法和通用矩阵分析法。波特五力模型和宏观环境分析都属于企业外部环境分析的方法。SWOT分析是通过对外部环境条件和内部资源能力的分析，进行系统评价，来指导企业进行战略选择和制定，它不属于投资组合分析法。

## 15.(单选题)下列选项中，不属于价值链分析法中基本活动的是（ ）。
* A 原材料搬运
* B 产品调适
* C 向供货商退货
* D 技术开发

正确答案：**`D`**

>解析
>
>价值活动包括基本活动和辅助活动。基本活动包括进货后勤、生产作业、发货后勤、经营销售和服务。辅助活动包括企业基础设施、人力资源管理、技术开发和采购。向供货商退货属于进货后勤，原材料搬运属于发货后勤，产品调适属于服务。技术开发属于辅助活动。

## 16.(单选题)下列关于价值链分析法的描述，错误的是（ ）。
* A 按照价值链分析法可以将企业活动细分为上千项
* B 对于经营者来说，所有活动都要创造价值，但它们在价值含义上的意义是不一样的
* C 价值活动的基础是企业资源
* D 不同行业的企业价值活动的侧重点是不一样的

正确答案：**`B`**

>解析
>
>B选项错误，价值是由企业所有活动创造的。对于经营者来说，所有活动都要创造价值，它们在价值含义上的意义是一样的。

## 17.(单选题)在SWOT模型中，具有一定的内部优势，但外部环境存在威胁企业，应采取（ ）。
* A 增长型战略
* B 多种经营战略
* C 扭转型战略
* D 防御型战略

正确答案：**`B`**

>解析
>
>在拥有外部威胁和内部优势的条件下，企业可以采用下面三种战略选择：第一，通过对外部环境施加影响，将威胁转化为机会。这只适合于非常有实力的企业，如中国石油、中国移动通过影响立法禁止民营企业进入。第二，将经营重点转移到其他有利可图的市场上（即多元化经营）。如联想公司将一部分资源转移到手机制造和销售。第三，企业可以利用自有资源和能力开发全新的产品和服务，进行业务的创新。如IBM公司，在大型机市场日趋萎缩的情况下，开发了以计算机服务为重点的服务增值业务，大大提高了公司的利润水平。

## 18.(单选题)价值链分析是应用于（ ）的工具。
* A 分析公司外部环境
* B 仅对公司内部分析，不关心竞争对手的情况
* C 发现公司有价值的资源和能力
* D 决定公司外部环境的机会能够维持多久

正确答案：**`C`**

>解析
>
>波特教授认为，企业每项生产经营活动都是其创造价值的经济活动，企业所有的互不相同但又相互关联的生产经营活动，构成了创造价值的一个动态过程，即价值链。用价值链分析法研究和理解企业怎样建立竞争优势非常有用。想要建立竞争优势就要发现自己有价值的资源和能力，所以答案选C。

## 19.(单选题)在SWOT模型中，处于第二象限的是（ ）。
* A 具有很好内部优势和外部环境的企业
* B 面临巨大外部机会。但内部资源匮乏的企业
* C 具有一定内部优势，但外部环境状况不佳的企业
* D 内外处境都令人堪忧的企业

正确答案：**`B`**

>解析
>
>第Ⅰ象限的企业，具有很好的内部优势以及众多的外部机会，应当采取增长型战略，如开发市场、增加产量等。
>第Ⅱ象限企业，面临巨大的外部机会，却受到内部劣势的限制，应采用扭转型战略，充分利用环境带来的机会，设法清除劣势。
>第Ⅲ象限企业，内部存在劣势，外部面临强大威胁，应采用防御型战略，进行业务调整，设法避开威胁和消除劣势。
>第Ⅳ象限企业，具有一定的内部优势，但外部环境存在威胁，应采取多种经营战略，利用自己的优势，在多样化经营上寻找长期发展的机会。

## 24.(单选题)下列属于潜在资源强势和竞争能力方面的指标的是（ ）。
* A 战略强大，有关键领域内的技能和专门技术的支持
* B 客户群会扩大或者业务扩张进入新的地域市场或产品细分市场

## 40.(论述题)请从下列几方面对直接利益相关者进行论述。  
（1）企业对股东的责任  
（2）企业对员工的责任  
（3）企业对消费者的责任

正确答案：

>（1）企业对股东的责任：

**·①企业要对股东的资金安全和收益负主要责任。股东把自己的资金托付给企业，并不是捐赠给企业，而是希望通过企业的经营获得丰厚的回报，企业应当感谢股东的信任和满足股东这个基本的期望。企业不得拿着股东的钱去做违法的、不道德的事情，企业更不能用股东的钱任意挥霍，企业所从事的任何经营活动必须以能给股东带来利润为基本前提。`** 

**`②企业有责任向股东提供真实的经营和投资方面的信息。企业应该通过企业年报、财务报表、大会等渠道及时向股东提供企业经营的各种信息，并保证公布的信息的真实性、可靠性，任何瞒报、谎报企业信息，欺骗股东的行为都是违法和不道德的，企业对此要负道德和法律的双重责任。  `**

>（2）企业对员工的责任：

**`企业与员工之间最基本的关系是建立在契约基础上的劳动关系，这种关系既包含法律关系，也包含道德关系。它是以劳动者提供劳动，企业提供薪酬和法定的福利为交换条件的雇佣关系；这种关系首先受我国的《劳动法》和《劳动合同法》保护，是一种劳动法律关系。同时由于劳动者与企业之间交换的不是普通的商品而是劳动，企业对员工之间的平等、尊重、信任、互助，以及企业对员工的心理、生活和职业生涯发展等都负有道德上的责任。  `**

>（3）企业对消费者的责任：

**`在社会分工越来越细的今天，每一个在社会里生活着的人都是企业的消费者或潜在的消费者。因此，从广义上说，整个社会成员都是企业的消费者。所以企业对消费者负责在某种意义上就是对社会负责的体现，企业对消费者的责任就是企业的社会责任。`**

## 41.(论述题)请从下列几方面对间接利益相关者进行论述。

**`在现代社会，企业对政府的责任主要表现为合法经营、照章纳税，这既是企业的法律责任，也是企业的社会责任。这是企业作为“社会公民”应尽的最基本的社会责任。企业是社会财富的创造者，政府是社会财富的管理者。政府通过价格、税收和福利政策实施社会财富的公正分配。企业合法经营、照章纳税是主动承担社会责任的体现；企业见利忘义、投机钻营、偷税漏税这些不良行为是对社会责任的逃避。  `**

>（2）企业对社区的责任：

**`企业是人与物的有机结合，所以企业一定会和周边的企业、社会组织、居民存在一定的关系。企业所在的周边环境统称为社区环境。所以企业与周边环境的关系又称为社区关系，建立和谐的企业与社区关系不仅对企业的生存发展有重要意义，而且对社区的进步繁荣具有重要意义。  `**

>（3）企业对环境的责任：

**`进入20世纪以来，企业成为科学技术飞速发展的受益者，同时企业也成为环境污染的主要角色，为此，企业在消除环境污染、保护环境中肩负着不可推卸的责任。企业对环境的首要责任体现为：树立人与自然和谐的价值观，努力做到尊重自然、爱护自然、合理地利用自然资源；其次，企业要以绿色价值观为指导，强化绿色环保意识，实施绿色管理，积极倡导绿色生产和绿色消费。`**

## 45.(论述题)试述企业对环境的责任。

## 47.(论述题)请从下列几方面阐述对企业价值链的理解。  
（1）价值链是可以不断细分的  
（2）价值是由企业所有活动创造的  
（3）各行业各企业价值活动的侧重点不同  
（4）价值活动的基础是企业资源

正确答案：

**`（1）价值链是可以不断细分的。按照价值链分析法可以将企业活动细分为几百上千项。如在技术开发方面，PC机的研发活动可以分为：外观、易用性、机器运行速度、兼容性、专用性等方面。在市场营销方面的活动可以分为：品牌、渠道、定价等等。  `**

**`（2）价值是由企业所有活动创造的。对于经营者来说，所有活动都要创造价值，它们在价值含义的意义是一样的。传统的观点把注意力只放在对基本活动的部分进行分析的方法是不全面。目前，在国有企业改制中，经常出现在同样的工人、同样的设备条件下，换了一个管理层，企业业绩就有了很大的提高的现象，这就是辅助活动带来了价值的增值。  `**

**`（3）各行业各企业价值活动的侧重点不同。不同行业的企业价值活动的侧重点是不一样的，如芯片制造业侧重于技术开发和制造，而零售企业则侧重在内部后勤、市场营销方面。同一行业中不同企业的价值活动的侧重点也是不一样的，如同样是PC机的制造，苹果公司与IBM公司，一个侧重于人性化设计，一个侧重于技术质量。  `**

**`（4）价值活动的基础是企业资源。价值活动的基础是企业资源，而企业资源是相对稳定的，所以价值活动也具有一定的稳定性，不能随意改变，这就形成了不同企业价值活动的特点。如海尔公司在品牌、管理等价值活动方面形成的优势是长期而稳定的，这属于市场营销和基础结构活动。但在技术开发和生产活动方面，海尔的优势并不明显，专利数也许不如海信，在成本方面则不如科龙等。`**

48.(论述题)请从下列几方面论述SWOT分析法的步骤  
（1）全面扫描业务  
（2）根据不  
同业务选择SWOT  
（3）战略匹配与选择  
（4）战略调整

正确答案：

**`（1）全面扫描业务。由于在SWOT要素识别时不同业务所对应的评价结果不同，并且不同业务对于企业发展的重要性不同，因此首先要做的是从企业整体入手，理清各业务范围，此时，要尽可能开放地列举业务，为以后的分析提供更多的线索，从而有利于开发出创新性想法或战略，这对企业而言可能更有价值。 `** 

**`（2）根据不同业务选择SWOT。针对第一步选择的业务，分别选择S、W、O、T，在选择时建议在SWOT后面都要标明具体对应的不同业务，使得识别出的各要素与不同业务的关系清晰明确，加强对业务的针对性。  `**

**`（3）战略匹配和选择。在上面步骤的基础上，进行S、W、O、T之间的关联与匹配，在这个步骤中，主要考察企业的SWOT匹配，判断企业业务处于何种组合。在这个过程中，可将时间因素考虑进去。由此根据标准的SWOT分析得出在OS、OW、TS和TW四种条件下的战略选择，明确各战略含义。  `**

**`（4）战略调整。得出SWOT结论后并不能一劳永逸，还要随时审视企业环境变化及各战略建议的合理性，进行调整。`**

## 49.(论述题)从下列几方面论述SWOT分析法战略匹配和选择。  
（1）针对OS匹配适宜选择的战略  
（2）针对OW匹配适宜选择的战略  
（3）针对TS匹配适宜选择的战略  
（4）针对TW匹配适宜选择的战略

正确答案：

**`（1）针对OS匹配适宜选择的战略：在拥有外部机会和内部优势的条件下，常采用的战略选择是增长战略。具体表现为：增加资源投入，巩固竞争优势，扩大对机会的利用，必要时需要企业在一定程度上集中所能支配的资源等。  `**

**`（2）针对OW匹配适宜选择的战略：第一，加强投资，将劣势转化为优势以开拓机会。由于企业处于劣势之中，一般要求企业将所有资源予以集中才可以改善实力。第二，在市场有吸引力的情况下，将业务放弃给竞争对手。  `**

**`（3）针对TS匹配适宜选择的战略：在拥有外部威胁和内部优势的条件下，企业可以采用下面三种战略选择：第一，通过对外部环境施加影响，将威胁转化为机会。这只适合于非常有实力的企业。第二，将经营重点转移到其他有利可图的市场上（即多元化经营）。第三，企业可以利用自有资源和能力开发全新的产品和服务，进行业务的创新。  `**

**`（4）针对TW匹配适宜选择的战略：在处于外部威胁和内部劣势的条件下时，企业可采用下面战略选择：第一，采取放弃或撤退战略。这样可以避免更大的损失；若有可能，也可将该业务出售，回收一部分资产。第二，防御战略。将业务收缩到一个相对实力较强的市场，通过集中于更细分的市场而形成优势。`**


## 50.(论述题)请论述非伦理行为产生的原因。
正确答案：

>对非伦理行为产生的原因进行一些粗浅的分析归纳：  

**`（1）个人伦理素质和道德准则对于管理者的行为有着显著的影响。实践证明，个人是非感强烈、道德情操高尚、伦理素质好的管理者在企业经营管理活动中采取非伦理行为的可能性相对较低，特别是较少卷入自利行为和行贿受贿。 `**

**`（2）管理者重视经济利益，忽略伦理判定。许多企业非伦理行为的出现，是因为管理者未能意识到自己行为的非伦理性，他们对伦理管理缺乏基本的认知，他们被灌输的是企业的存在就是为了追求利润最大化。因此，他们计算的只是商业利益，并认为企业战略是单纯的商业决策，而忽视了这一决策的伦理影响。  `**

**`（3）管理者不注重企业文化的建设，不知道企业文化的核心是伦理观，在决策中只考虑经济因素。  `**

**`（4）高层管理者对绩效目标的不现实的要求，除非走捷径或违反伦理，否则根本做不到。在此压迫下，中层管理者放弃原则和标准。  `**

**`（5）非伦理领导。领导在组织文化的建立中起到重要的作用，他们提供了榜样的示范。组织内的员工常 常从企业领导身上获得暗示，如果这些领导的行为不符合伦理，员工也会有样学样。`**
